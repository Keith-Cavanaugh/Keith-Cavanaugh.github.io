The Space Racers

Space Racer is a time trial racing game built to demonstrate “Game Feel.” That is the elusive yet essential quality that makes certain games more fun to play. We built Space Racer to be fast paced but with tight controls to achieve a greater sensation of involvement with the game.

Working on this project was an opportunity to try new programs and languages. I enjoyed working in Unity 3D and learned C# and Javascript to write in game actions. I welcomed taking on the many roles required to pull together a video game, and the rewards it can provide in return.

This was the first time I actually drew upon physics in my computer science work, utilizing formulas to guide my avatar and recursive functions to generate realistic asteroid fields. This project was a great way to combine a passion for programming, graphics, and user experience.

Bread Breaker

Prototype for User Interface Design

For this design, our team decided to focus on the implementation of a wearable companion app. Most people do not “research” restaurants in depth before eating there; they simply ask a friend. We believed that we better captured the spirit of this interaction through our user based development.

This project allowed me to evaluate computing related problems, develop interface design alternatives, prototype software, and analyze the results. Our application was successful as a result of extensive testing that allowed us to accurately judge user characteristics and needs. Most importantly, I learned that design is a systematic and data driven process.

The Space Racers

Space Racer is a time trial racing game built to demonstrate “Game Feel.” That is the elusive yet essential quality that makes certain games so fun to play. We built Spacer Racer to be fast paced but with tight controls to achieve this sensation of involvement with the game.

Working on this project was an opportunity to try new programs and languages. I liked working in Unity 3D and learned C# and Javascript to write in game actions. I enjoyed taking on the many roles it takes to pull together a video game, and the rewards it can provide in return!

For the first time, I was using physics in my work! I needed formulas to guide my avatar and recursive functions to generate asteroid fields. I found this project a great way to combine a love for programming, graphics, and user experience.
